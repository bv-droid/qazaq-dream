# Qazaq Dream

Фильм-манифест: дом перестаёт быть делом жизни и становится местом для жизни.

Одна страница. Прокрутка ведёт фильм из 3481 кадра — от степи и аула
до квартала таунхаусов и обратно к рисунку ребёнка. Заканчивается знаком.

**Смотреть:** https://bv-droid.github.io/qazaq-dream/

## Устройство

- `index.html` — вся страница: разметка, стили, движок. Кривые знака и ворднейма
  вшиты в файл, поэтому страница открывается и без сервера.
- `assets/film/` — кадры фильма, 12 к/с, 1024 px.
- `assets/plate.png` — кадр для карандашного наброска на стартовом экране.
- `assets/favicon.svg`, `assets/og.jpg` — иконка и карточка для мессенджеров.

Движок: кадры рисуются на canvas, индекс кадра считается от прокрутки
с инерцией. Опорные кадры грузятся сразу, остальные — окном вокруг текущего,
дальние освобождаются. Тексты привязаны к номерам кадров, а не к процентам
прокрутки, и держатся заданное число кадров — читаются при любой скорости.

Языки: қазақша (по умолчанию) и русский, переключатель в правом верхнем углу.
